# AD Group Export Tool

Инструмент с графическим интерфейсом для экспорта участников группы Active Directory в CSV-файл.  
Программа использует PowerShell и модуль ActiveDirectory для получения данных и позволяет гибко настраивать параметры экспорта.

![Скриншот](screenshot.png)

## Возможности

- Экспорт пользователей из указанной группы AD
- Рекурсивный обход вложенных групп (опционально)
- Выбор кодировки CSV: UTF-8, UTF-8 с BOM, ASCII, Unicode
- Добавление данных в конец существующего файла (append)
- Предварительный просмотр первых строк экспортированного файла
- Копирование текста из окна результатов (правой кнопкой мыши)

## Требования

- **Операционная система:** Windows (из-за использования PowerShell и модуля AD)
- **PowerShell** версии 3.0 или выше (входит в состав Windows)
- **Модуль ActiveDirectory** для PowerShell (часть RSAT, устанавливается отдельно)
- **Python** версии 3.6+ (если запускать из исходного кода)

## Установка и запуск

### Запуск скрипта

1. Убедитесь, что Python установлен и добавлен в `PATH`.
2. Дважды кликните на файл export-pbi.pyw
